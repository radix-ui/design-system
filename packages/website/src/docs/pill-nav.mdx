### Custom props

| Name        | Type            |   Default   | Description                           |
| :---------- | :-------------- | :---------: | :------------------------------------ |
| `active`    | Boolean         |   `false`   | Changes the syling to active state    |
| `disabled`  | Boolean         |   `false`   | Disables the element                  |
| `pillWidth` | ResponsiveValue | `undefined` | Sets a fix width to the Pill elements |
