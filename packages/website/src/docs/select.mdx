## Examples

### Size

There are 2 sizes to choose from.

```js live
<Select>
  <option>Option 1</option>
  <option>Option 2</option>
</Select>
<Select mt={6} size={1}>
  <option>Option 1</option>
  <option>Option 2</option>
</Select>
```

### Variants

There are 2 variants to choose from.

```js live
<Select variant="normal">
  <option>Option 1</option>
  <option>Option 2</option>
</Select>
<Select mt={6} variant="ghost">
  <option>Option 1</option>
  <option>Option 2</option>
</Select>
```
