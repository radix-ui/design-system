Radix is a Component Library built on top of [Radix System](https://github.com/modulz/radix/tree/master/packages/radix-system).

<Divider size={2} my={6} />

### Install it

Radix is currently under heavy development, but if you'd like to use it anyway:

```js
# NPM
npm install @moduz/radix
```

```js
# Yarn
yarn add @moduz/radix
```

### Use it

All Components can be importated directly:

```js
import { Heading, Text, Link } from '@radix/modulz';
```

### Wrap it in `RadixProvider`

Make sure to wrap your app in `RadixProvider` to get the default theme

```js
import { RadixProvider } from '@modulz/radix';
```

### Build stuff with it

```js live=true
<RadixProvider>
  <Box bg="grays.1" p={4}>
    <Heading mb={4} fontWeight="bold">
      Composing with Radix
    </Heading>
    <Text as="p">Browse our list of components to find out more about each one.</Text>
    <Divider size={1} my={4} />
    <Text as="p" size={2}>
      Built by the <Link href="https://modulz.app">Modulz</Link> team.
    </Text>
  </Box>
</RadixProvider>
```
