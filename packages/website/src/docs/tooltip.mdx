### Custom props

| Name                 | Type                                   | Default  | Description                     |
| :------------------- | :------------------------------------- | :------: | :------------------------------ |
| `side`               | 'top' \| 'bottom' \| 'right' \| 'left' | `bottom` | Changes the side                |
| `sideOffset`         | nunber                                 |   `-5`   | Changes the side offset         |
| `align`              | 'start' \| 'end'                       | `start`  | Changes the alignment           |
| `alignOffset`        | number                                 |   `0`    | Changes the alignment offset    |
| `arrow`              | React.ReactNode                        | `Arrow`  | Changes the arrow component     |
| `arrowOffset`        | number                                 |   `0`    | Changes the arrow offset        |
| `collisionTolerance` | number                                 |   `0`    | Changes the collision tolerance |
| `label`              | string                                 |          | Sets the label                  |
| `ariaLabel`          | string                                 |          | Sets the arial-label            |
| `children`           | React.ReactNode                        |          | Sets the chilren                |
| `css`                | CSSProperties                          |          | Changes the CSS                 |
